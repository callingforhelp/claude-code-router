# Claude 3.5 Test Script

This is a simple test script that demonstrates various capabilities using Python. The script includes:

- Type hints and documentation
- Logging configuration
- Class-based design
- Text processing functionality
- Numerical analysis
- Error handling
- JSON output formatting

## Structure

- `test_claude.py`: Main Python script with test functionality
- `requirements.txt`: Project dependencies (none required - uses standard library)

## Running the Tests

To run the test script:

```bash
python test_claude.py
```

The script will output:
1. Text processing statistics for a sample sentence
2. Statistical analysis of a sample number list

## Expected Output

You should see JSON-formatted output showing:
- Word count, character count, and average word length for the sample text
- Mean, min, max, and range for the sample numbers

The script also includes logging that will show when the processor is created and when text is being processed.
